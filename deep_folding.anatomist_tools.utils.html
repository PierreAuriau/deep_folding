deep_folding.anatomist_tools.utils.load_bbox module¶

Getting bounding box from a list of sulci and specific hemisphere

The aim of this script is to output the bounding box of one or several sulci of a specified hemisphere.

deep_folding.anatomist_tools.utils.load_bbox.compute_max_box(sulci_list, side, talairach_box=False, src_dir='/neurospin/dico/deep_folding_data/data/bbox/')¶

Function returning maximal bounding box of a given list of sulci

It reads json files contained in the source directory. They are listed in subdirectory ‘L’ or ‘R’ according t hemisphere Each json file is the name of the sulcus + ‘json’

Parameters

  • sulci_list – a list of sulci

  • side – a string corresponding to the hemisphere, whether ‘L’ or ‘R’

  • talairach_box – a boolean whether using Talairach coordinates or voxels

  • src_dir – path to source directory containing bbox dimensions

Returns

an array of minimum coordinates of bounding box of given sulci bbmax: an array of maximum coordinates of bounding box of given sulci

Return type

bbmin

deep_folding.anatomist_tools.utils.resample module¶

Resample a volume that contains discret values

deep_folding.anatomist_tools.utils.resample.resample(input_image, transformation, output_vs=None, background=11, values=None, verbose=True)¶

Transform and resample a volume that as discret values

Parameters

  • input_image (file) – Path to the input volume (.nii or .nii.gz file)

  • transformation (file) – Linear transformation file (.trm file)

  • output_vs (tuple) – Output voxel size (default: None, no resampling)

  • background (int) – Background value (default: 11)

  • values ([]) – Array of unique values ordered by descendent priority. If not given, priority is set by ascendent values

Returns

Transformed and resampled volume

Return type

resampled_vol

deep_folding.anatomist_tools.utils.sulcus_side module¶

Getting complete sulci names based on generic name and side

The aim of this script is to output the complete name of sulci. For instance, giving “S.T.s.ter.asc.ant” and right side, script outputs “S.T.s.ter.asc.ant._right”

deep_folding.anatomist_tools.utils.sulcus_side.complete_sulci_name(sulci_list, side)¶

Function gathering sulci and side to obtain full name of sulci

It reads suli prefixes from a list and adds a suffix depending on a given side.
